First Block

Create a 3x2x1 block on the Left slice. Following the example from Roux, I've been using White on the Down face and Green on the Left. So this means solving the block consisting of the Green centre, the Green/White edge, the Green/Red edge, the Green/Orange edge, the Green/White/Orange corner and the Green/White/Red corner. (Diagram looking at the L-face) Hint for speed: Try to plan out the whole of this step in the inspection phase and to execute the moves without looking.
